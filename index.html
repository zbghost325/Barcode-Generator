📊 How to Use

Preparing Your Data:

  1. Open your Excel spreadsheet with asset data
  2. Arrange your data in 3 columns:
    Column A: Device Name
    Column B: Serial Number
    Column C: Asset Tag
  3. Select all the data (including all 3 columns)
  4. Copy the selection (Ctrl+C or Cmd+C)

Choosing a Label Format:

  • Standard Barcode: Full-size label with device name, serial number, Code-39 barcode, and asset tag number. One device per label.
  • Strip Labels (2 per label): Each Avery 5520 label is split into two thin horizontal strips. Each strip has SN and AT QR codes plus the device name. Ideal for XR glasses, thin devices, or narrow surfaces.

Generating Labels:

  1. Select your label format (Standard or Strip)
  2. Paste your data into the text box
  3. Click "Generate Labels"
  4. Review the preview to ensure correct formatting
  5. Click "Print on Avery 5520" when ready

🖨 Print Settings

⚠ Important: Use these exact settings for proper alignment

Required Print Settings:

  1. Scale: Set to 100% (no scaling/shrinking)
  2. Margins: Choose "Default" or "None"
  3. Headers/Footers: Turn OFF (uncheck this option)
  4. Paper Size: Letter (8.5" x 11")
  5. Orientation: Portrait
  6. Color: Black & White is fine for barcodes

🔧 Troubleshooting

Labels Not Aligning:

  • Verify scale is set to exactly 100%
  • Check that margins are set to "Default" or "None"
  • Ensure headers/footers are disabled
  • Try printing a test page on regular paper first

Barcodes Not Scanning:

  • Ensure asset tags contain only numbers and letters
  • Check printer quality settings (use "Normal" or "Best")
  • Verify the scanner supports Code-39 format
  • Clean the printer heads if bars appear faded

Strip Label QR Codes Not Scanning:

  • Use a QR code scanner app or camera
  • Ensure good lighting and print quality
  • QR codes encode the serial number and asset tag as text

Multi-Page Printing:

Note: When printing more than 30 labels, page 1 is optimized for perfect alignment. Pages 2+ use standard spacing. For best results with multiple pages, consider printing in batches of 30.

💡 Tips & Best Practices

Pro Tips:
  • Always print a test page on regular paper first
  • Keep serial numbers consistent in format
  • Use sequential asset tags for easier tracking
  • Save your Excel data for reprinting if needed
  • Strip labels are great for XR glasses, styluses, and thin devices
  • Cut strip labels along the dashed line with scissors or a paper cutter

📝 Technical Specifications

Barcode Details:

  • Standard Mode: Code-39 (3 of 9)
  • Strip Mode: QR Code (ISO/IEC 18004)
  • Character Set (Code-39): A-Z, 0-9, and special characters (- . $ / + % space)
  • QR Capacity: Up to 4,296 alphanumeric characters

System Compatibility:

  • Operating Systems: Windows 10/11, macOS, Linux
  • Browsers: Chrome 90+, Edge 90+, Firefox 88+, Safari 14+
  • Printers: Any printer supporting Letter size paper
  • Scanners: Any Code-39 or QR code compatible scanner
